## 145. Binary Tree Postorder Traversal

`Easy`

Given the <code>root</code> of aï¿½binary tree, return <em>the postorder traversal of its nodes' values</em>.

---

### Examples


![](pre1.jpg)

**Input**
```
root = [1,null,2,3]
```

**Output**
```
[3,2,1]
```

---

**Input**
```
root = []
```

**Output**
```
[]
```

---

**Input**
```
root = [1]
```

**Output**
```
[1]
```

---

**Constraints**

<ul>
<li>The number of the nodes in the tree is in the range <code>[0, 100]</code>.</li>
<li><code>-100 &lt;= Node.val &lt;= 100</code></li>
</ul>
